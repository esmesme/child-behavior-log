# Child Behavior Log Plugin

Child behavior logger for emotionally dysregulated interactions.

A simple, thoughtful tool for parents navigating complex family dynamics. Designed with DBT-inspired principles in mind, this plugin helps you log daily interactions with your child in a way that’s quick, non-judgmental, and insightful.

## Features
*   **Quick 5-question log**: context, behavior, emotions, thoughts, and response
*   **Built-in stressor checkboxes** (tired, hungry, overwhelmed, etc.)
*   **Lightweight and private** — all data stays in your vault
*   **Great for therapy prep or personal reflection**

## Why It Helps
Perfect for parents who want to:

*   Stay calm and reflective during tough moments
*   Spot patterns over time
*   Communicate more clearly with therapists

This plugin was built with love to help one mom feel more grounded. Now it’s ready to help others too.
